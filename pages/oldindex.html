Kingdom Keys Re:Coded is a modification for Minecraft currently developed by Toby "Wehavecookies56" Wishart and Abelatox. The latest version is v0.7.6 for Minecraft 1.8.9 and is developed using Minecraft Forge. It is also open source located on Github.
